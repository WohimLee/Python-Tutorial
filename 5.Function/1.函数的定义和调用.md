# 函数
# 1 定义
- 你可以定义一个由自己想要功能的函数，以下是简单的规则：

    - 函数用 `def` 关键词定义，后接 `函数名` 和 `圆括号 ()`
    - 圆括号内可以设置参数
    - 函数体内通常会有一些注释说明参数、返回值。
    - 函数内容以冒号 : 起始，并且缩进。
    - return [表达式] 结束函数，选择性地返回一个值给调用方，不带表达式的 return 相当于返回 None。


>语法

- Python 定义函数使用 def 关键字，一般格式如下：
    ```python
    def 函数名(参数1: 参数类型, 参数2: 参数类型, ..., *args, **kwargs)->返回值:
        '''
        函数说明
        '''
        函数体
        return # 可选
    ```

>示例
```python
# 以下 3 种形式等价
def hello() :
    print("Hello World!")

def hello() :
    print("Hello World!")
    return 

def hello() :
    print("Hello World!")
    return None
```

# 2 函数调用
- 直接使用函数名，传入参数
- 有返回值的可以赋值给其他变量，或者直接打印出来
>示例
```python
def max(a, b):
    if a > b:
        return a
    else:
        return b
 
a = 4
b = 5
print(max(a, b))

def area(width, height):
    return width * height
 
def print_welcome(name):
    print("Welcome", name)
 
print_welcome("aaaa")
w = 4
h = 5
print("width =", w, " height =", h, " area =", area(w, h))
```
